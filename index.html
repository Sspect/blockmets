ℹ️ Information about the Server

⚠️ Warning: Audio version may be loud.

Connect to the server by doing the following:

1. Get whitelisted by joining the Oslomet Discord and following the instructions in the minecraft chat channel.

2. Use a Java Minecraft client running version 1.21.8.

3. Connect to the server using the server address survival.blockmets.no

🛠️ Non-Vanilla Features (No Mods Needed!)

The server uses server-side datapacks and mods to add extra features. You do not need to install any mods to join—just use a regular Minecraft client!

  • Decorative & Cosmetic:
    • Player head drops (video)
    • Mini blocks
    • Armor statues (video)
  • Convenience:
    • Dragon drops Elytra and Dragon Egg
  • Gameplay Changes:
    • Husks drop sand
  • Informative:
    • Villager death messages (video)
  • Vaulted End:
    Vaulted End is a server-side mod that replaces Elytra item frames in End City Ships with Vaults and Trial Spawners. This helps new players get Elytra and Shulker Shells even in an old End, without needing to travel far.
  • Disable End:
    Adds a gamerule to disable end portals. The End will open on a specific date so everyone can cooperate on killing the dragon together.
  • Universal Shops:
    Flexible, server-side trade shop mod. Works with vanilla clients—no mods needed! More info: Universal Shops
  • Hey That’s Mine:
    Protect your containers (chests, furnaces, etc.) from theft. You can trust other players to access your containers. See commands below for how to use it.

📌 Useful Commands on the Server

Try typing these in Minecraft chat

🌐 Web & Maps

/web → Website

/map1 → Chunkbase Seed Map

/map2 → BlueMap Live Map

🏠 Homes

/sethome [name] → set a new home

/home [name] → teleport to a home

/delhome [name] → delete one of your homes

/homes → list your homes

⚠️ Each player can set up to 3 homes.

🚪 Warps

/warp spawn → teleport to the spawn point

/warp games → teleport to the games area

/warp shops → teleport to the shops area

🌀 Teleportation

/tpa [player] → request to teleport to a player

/tpahere [player] → request a player to teleport to you

💀 Graves

Grave boxes last for 15 minutes before they break (items drop on the floor).

/graves → allow other players to access your grave box

🔒 Hey That’s Mine

Docs: modrinth.com/mod/htm
Containers locked by default include: chests, trapped chests, and furnaces.

/htm trust [name] global → trust a player for all your containers

/htm set PUBLIC → (next left-click) make a container public

/htm set PRIVATE → (next left-click) make a container private

/htm info → (next left-click) show info about a container

/htm persist → repeat the same action on multiple containers

🎤 Voice Chat

Docs: voicechat wiki / commands

/voicechat → basic settings

/voicechat test → test your mic

ℹ️ Voice Chat is optional — you don’t need it to join the server.

🎲 Fun Commands

/hat → wear the item in your hand as a hat 🧢

/random roll 1..6 → roll a dice 🎲

📜 Server Rules

⚠️ Warning: Audio version may be loud.

These rules may be adjusted as needed!

  1. Respect All Players
    No harassment, hate speech, or bullying. Treat everyone with kindness.
  2. No Griefing
    Don’t destroy or steal from other players’ builds or property.
    Staff can track block breaking and items taken from chests.
  3. Protect the Environment
    Don’t cause excessive damage to the landscape (e.g., avoid leaving floating trees).
  4. Follow Staff & Appeal Process
    Helpers and admins have the final say in-game. Respect their decisions.
    Not doing so may result in warnings, temporary bans, or permanent removal.
    If you disagree with a decision, you can appeal to OMG Board members.
  5. Report Issues
    If you see someone breaking the rules, report it to a helper, admin, or board member.
  6. No Cheating or Hacking
    Do not use hacked clients, x-ray, duplication glitches, or any mods that give an unfair advantage.
    Breaking this rule will result in a ban.

    ✅ Allowed client-side mods:
    • Lightmatica
    • Simple autoclickers (not combat hacks)
    🟥 Not allowed:
    • X-ray or ore-finding mods
    • Fly/Speed hacks
    • Anything that automates combat or gives unfair PvP advantages
  7. How to Get Whitelisted
    To get whitelisted, join the Oslomet Discord and follow the instructions in the minecraft chat channel.

⬆️ If you have any suggestions, post them in Minecraft chat!
